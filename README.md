# Campus Event Management System

Bu proje, kampüs içerisindeki öğrenci, kulüp ve etkinlik yönetimini sağlayan modüler bir Python uygulamasıdır.

## Özellikler
- Öğrenci ekleme ve silme
- Etkinlik oluşturma
- Oda–tarih–saat çakışma kontrolü
- Etkinlik kayıt sistemi
- Lambda fonksiyonları ile raporlama

## Çalıştırma
Terminalden aşağıdaki komut ile çalıştırılır:

python demo.py
